# 제로스트 · 마녀스프 MVP 기획서

> Notion에 붙여넣기용. 최종 업데이트: 2026-05

## 1. 한 줄 컨셉

일상 **제로 미션**을 하면 **재료**가 쌓이고, 재료 4개를 냄비에 넣어 **마녀스프**를 끓이면 **에코잼(인게임)** 또는 **실물 리워드(히든)** 를 얻는다.

## 2. 목표

- 기능 최소 · 체감 가벼움
- **리워드는 화면에서 확실히** (에코잼 / 실물)
- 주간 레시피 + 히든 1회 제작으로 **재방문**

## 3. 하단 네비 (4탭)

| 탭 | 역할 |
|---|---|
| 재료 | 보유 재료 · **슬롯 4칸** · 스프 만들기 |
| 홈 | 마녀(고양이) + 냄비 · **오늘의 레시피 힌트** · 미션 진입 |
| 레시피 | 주간 일반(공개) · 히든(??? ) |
| 마이 | 에코잼 · 완성 레시피 · 실천 기록 |

## 4. 재료창 UX

```
[ □ ] [ □ ] [ □ ] [ □ ]   ← 상단 고정 4칸
─────────────────────
(보유 재료 그리드, 탭 시 빈 칸에 순서대로 채움)
칸 탭 → 해당 재료 제거
─────────────────────
[ 스프 만들기 ]  ← 4칸 모두 채워지면 활성
```

- 보유 수량 0인 재료는 비활성
- 조합이 **이번 주 일반 레시피** 또는 **히든(미완료)** 와 일치하면 제작 가능
- **각 레시피는 1회만** 제작 가능 (완료 후 재도전 불가)

## 5. 레시피 규칙

| 구분 | 변경 주기 | 노출 | 보상 |
|---|---|---|---|
| **일반** | **매주** 교체 | 이름·재료·약한 보상 공개 | **에코잼** (인게임) |
| **히든** | **고정** | 미해금 시 `???` | **실물 리워드** (쿠폰·굿즈 등, 기획 확정) |

## 6. 홈 (마녀스프)

- 중앙: 마녀(또는 마녀 고양이) + 냄비(저어지는 연출은 Phase 2)
- **오늘의 레시피 힌트** 1줄
- CTA: **오늘 미션 하기** → 기존 미션 플로우

## 7. 미션 → 재료

- 미션 완료 시 **재료 1종** 지급 (미션별 매핑)
- 토스트: 「○○ 재료를 얻었어요」

## 8. 스프 만들기 결과

- 성공: 연출 + 레시피명 + **에코잼 +N** 또는 **실물 리워드 안내**
- 이미 만든 레시피 조합 → 「이미 만든 스프예요」

## 9. 비주얼

- 포인트 컬러: **보라 계열** (`#7C5CBF`, `#EDE7F6`)
- 톤: 앱인토스 해요체, 카드·여백 넉넉히

## 9. 온보딩

- 플로우 유지: 환영 → 샵 선택
- 비주얼만 보라 톤 + 마녀스프 세계관 카피

## 10. MVP 범위 (FE)

- [x] 4탭 골격
- [x] 재료 4슬롯 + 제작 + 결과 화면
- [x] 레시피 목록 (주간/히든)
- [x] mock 데이터 + Storage
- [ ] Lottie / 일러스트 (상민 시안 후)
- [ ] BE · 실물 리워드 연동
