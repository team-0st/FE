# 제로스트 파일럿 User Flow

> 최종 업데이트: 2026-07-10 · 파일럿 샵: **알맹상점 마포점**

## 1. 진입·온보딩

```
앱 실행
  └─ (최초) /onboarding — 새싹 인트로
       └─ /onboarding/profile — 닉네임·전화번호(010)·알맹 포인트 동의
            └─ /onboarding/shop — 단골 샵 선택 (파일럿: 알맹상점 마포점)
                 └─ / — 홈
```

- 재방문: `onboardingCompleted === true` 이면 홈으로 바로 진입
- 전화번호: 매장 포인트 지급·본인 확인용 (알맹 기존 방식과 동일 — 번호만 입력)

## 2. 일일 루프 (홈 중심)

```
홈 (/)
  ├─ 출석 카드 1탭 → POST /check-in (mock) → 랜덤 재료 1개
  ├─ 주간 미션 O/X 요약 → /missions
  ├─ 오늘의 레시피 힌트
  ├─ 커뮤니티 목표 (mock 진행률)
  └─ 주변 상점 미리보기 → /shop/partners (위치 동의 후)
```

### 미션 플로우

```
/missions — 일반·특별·공동 미션 목록
  └─ /missions/:id — 상세
       └─ /missions/:id/verify — 사진 인증
            └─ /missions/:id/result — 완료·검수 대기
                 ├─ (완료) SNS 공유 → 에코잼 보상 (하루 1회, mock)
                 └─ 홈으로
```

- 공동 미션: ⭐→⭐⭐→⭐⭐⭐ 해금 (파일럿 mock: ⭐1만 활성)
- 미션 완료 시 **재료 1종** 지급 → 제작 탭에서 사용

## 3. 주간 루프 (재료 → 스프 → 가챠)

```
/ingredients (제작 탭)
  ├─ 슬롯 3+1 (일반 3 / 히든·전설 4)
  ├─ 보유 재료 그리드
  └─ [스프 만들기] → POST /api/soup/craft (mock)
       └─ /soup/result — 보상 표시 + (성공 시) SNS 공유
```

```
/gacha — 에코잼 100개 소모 뽑기
  └─ 결과 카드 + SNS 공유 (하루 1회 보상 mock)
```

```
/recipes — 이번 주 레시피(공개) · 히든·전설(???)
/profile (마이) — 에코잼·알맹P·재료·미션·내역
```

## 4. 매장·포인트 연동 (오프라인)

| 단계 | 앱 | 매장 |
|------|-----|------|
| 적립 | 미션·스프·가챠 등으로 **알맹 포인트 적립** (앱 state) | — |
| 지급 | 동의 + 전화번호 등록 사용자 | 매장에서 **전화번호 입력**으로 본인 확인 후 지급 |
| 주변 안내 | `/shop/partners` 지도·리스트 (341곳 mock) | 파일럿 포인트 연동은 **알맹상점만** |

## 5. 하단 탭

| 탭 | 경로 | 역할 |
|----|------|------|
| 제작 | `/ingredients` | 재료·스프 제작 |
| 가챠 | `/gacha` | 에코잼 뽑기 |
| 홈 | `/` | 출석·미션·힌트 |
| 레시피 | `/recipes` | 주간·히든 목록 |
| 마이 | `/profile` | 재화·기록 |

## 6. BE 연동 시 교체 지점

- `UserProvider` — checkIn, verify, soup, gacha API
- `totalPoints` / `almangPayoutConsent` — 서버 원장 동기화
- 공동 미션·커뮤니티 목표 — 실시간 집계 API
