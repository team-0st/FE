### 진짜 브랜드 그림이 없는 이유

Figma 초안의 히어로(🌱🎁🍲🥕…)는 **텍스트 이모지**입니다.
이미지 파일로 심어 둔 커스텀 에셋이 아닙니다.
재료·캐릭터 PNG는 디자이너가 그려 `assets/brand/`에 넣는 흐름입니다.
